# `repo`

## `repo`

````cli-help
A tool for repo management.

Usage: repo <COMMAND>

Commands:
  version      Run a single search
  completions  Print completions for the given shell
  help         Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
````
